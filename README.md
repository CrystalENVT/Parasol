## About Parasol
Named after the [Parasol Mushroom](https://en.wikipedia.org/wiki/Macrolepiota_procera), Parasol is a self hosted bot for Stoat. Initial Scope / MVP for Parasol is "Channel Live Notification" (for Twitch And/Or YouTube) & Reaction Roles.

## How to run
Ensure Python 3.10 or greater is installed (required for Stoat.py dependency).  
Run the following the commands:
1) `python3 -m venv --upgrade-deps venv`
2) `source ./venv/bin/activate`
3) `pip3 install -r requirements.txt`
4) `python3 ./parasol.py`
